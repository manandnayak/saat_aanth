# Saat – Aanthh (Mobile PvC)

This is a Flutter project for the 2‑player card game **Saat – Aanthh**.

## Quick start

### 1) Install Flutter
- https://docs.flutter.dev/get-started/install

### 2) Run on an Android phone (USB debugging)
```bash
flutter pub get
flutter run
```

### 3) Build an installable APK (Android)
```bash
flutter build apk --release
```
The APK will be at:
`build/app/outputs/flutter-apk/app-release.apk`

Copy it to your phone and install it.

> iOS builds require Xcode on macOS and an Apple Developer account to install on a device.

## Notes
- The game logic follows the rules provided in chat.
- AI has 3 aggressive levels.
